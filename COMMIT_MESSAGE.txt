JaePa 핵심 기능 개선 및 향상

## 주요 변경사항

### 1. JWT 기반 인증 시스템 추가
- 토큰 발급, 검증, 갱신 기능 구현
- 사용자 관리 기능 (등록, 로그인, 권한 관리) 구현
- 역할 기반 접근 제어 구현

### 2. 크롤링 퍼포먼스 모니터링 시스템 구현
- 크롤링 작업 통계 수집 기능 추가
- 성능 지표 모니터링 구현
- 오류 및 경고 기록 시스템 구현

### 3. 크롤링 유효성 검증 시스템 추가
- 크롤링된 기사 구조 검증 기능 추가
- 날짜, URL, 내용 유효성 검사 구현
- 품질 점수 계산 로직 추가

### 4. 개선된 감성 분석 시스템
- FinBERT 모델을 활용한 감성 분석 기능 개선
- 신뢰도 지표 추가
- 제목과 본문의 감성 일관성 검증 구현
- 배치 처리 기능으로 대용량 데이터 처리 최적화

### 5. 자동화된 테스트 시스템
- 크롤러, 감성 분석기에 대한 단위 테스트 작성
- 통합
